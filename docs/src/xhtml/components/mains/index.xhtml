• Main

    If your app sections are more or less hardcoded, or if you are sketching a prototype design, the Main element may come in handy to generate Header tabs.

    The Main component is a fairly unassuming part of the basic layout where it functions to control the flex. Here it is, stuck between the App and Content sections.

    We can add multiple Main elements inside a containing Mains to automatically generate tabs in the Header. They must all be fitted with a label for this to work.

    The Header will automatically be updated when you add or remove Main elements just as long as you remember to declare the label attribute.
