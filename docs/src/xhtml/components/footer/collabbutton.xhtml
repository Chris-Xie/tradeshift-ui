Footer collabbutton

The Footer features a special button which is dedicated to launching the collaboration panel. To show the button, call collabbutton with a onclick method and a callback.

Notification state

The collaboration button can have two states: a default state and a notification state. The default state shows just the special button. The notification state shows a small notification badge over the icon. To show the button with the notification state, call the method with the onclick property and a callback, as show above, and also a notification property set to true.

Note that the Footer doesn't automatically initialize or resume the collbaroration. The API for this will be provided elsewhere. You can hide the collaboration button by passing null to the same method.
