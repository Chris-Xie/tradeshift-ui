# Basic Inputs

### Elements of an input field

Input fields have several options:

* Input value entry field and type (required)
* Label (required)
* Placeholder text
* Field required indication
* Multiline
* Input masking (not supported atm)
* Help text attachment
* Field validation error
* Picker enabled
* Disabled/locked
